Игра, разработанная мною на Unity в 2019, в стиле Lode Runner - аркадной головоломки, только в 3D. Геймплей представляет из себя хотьбу по лабиринту в поисках монет, которые в свою очередь открывают запертые двери. Цель - найти выход из лабиринта.
